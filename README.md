# Smart-Bin-Monitoring-System
IoT-based Smart Bin Monitoring System using ESP32 and Blynk.

## Overview
A smart IoT-based waste bin monitoring system developed using ESP32.

## Features
- Monitors waste level using an HC-SR04 ultrasonic sensor.
- Displays the bin status on an LCD screen.
- Sends real-time notifications using the Blynk application.
- Supports wireless monitoring through Wi-Fi.

## Hardware
- ESP32
- HC-SR04 Ultrasonic Sensor
- LCD Display
- Jumper Wires
- Breadboard

## Software
- Arduino IDE
- Blynk IoT Platform
